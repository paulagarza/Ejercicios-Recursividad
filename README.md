# Ejercicios-Recursividad
## Ejercicio 1: Suma de dígitos pares de un número (3 puntos)

Escribe una función estrictamente recursiva que reciba un número entero positivo n y devuelva la suma de sus dígitos que sean pares.

Ejemplo:
Si n = 48271 → devuelve 4 + 8 + 2 = 14

⚠️ No se permite convertir el número a string ni usar bucles.

Requisito extra:
Indica la complejidad temporal en Big-O.

## Ejercicio 2: Recursividad y Divide y Vencerás (3 Puntos)
Escribe una función estrictamente recursiva (no se permite el uso de ningún bucle for, while o do-while) que reciba un arreglo de enteros y su tamaño, y devuelva la suma de todos los números pares que se encuentren en posiciones impares del arreglo.
Requisito extra: Una vez resuelto, explica brevemente cuál es la complejidad temporal (Big-O) de tu solución.

## Ejercicio 3: Buscar si existe un elemento repetido consecutivo (3 puntos)

Implementa una función recursiva que reciba:

Un arreglo de enteros

Su tamaño

Y devuelva true si existen dos elementos consecutivos iguales.

Ejemplo válido:
[1, 5, 7, 7, 9] → true

Ejemplo no válido:
[1, 5, 7, 8, 9] → false

⚠️ Prohibido usar bucles.

Requisito extra:
Justifica la complejidad temporal.


## Ejercicio 5: Determinar si un arreglo es palíndromo (3 puntos)

Escribe una función estrictamente recursiva que reciba:

Un arreglo de enteros

Dos índices: inicio y fin

Y determine si el arreglo es palíndromo.

Ejemplo:
[1, 2, 3, 2, 1] → true
[1, 2, 3, 4, 1] → false

⚠️ No se permiten bucles.

Requisito extra:
Indica la complejidad temporal y espacial (por la pila de recursión).


## Ejercicio 6: Contar ocurrencias de un dígito en un número (3 puntos)

Escribe una función estrictamente recursiva que reciba un entero positivo n y un dígito d (0–9) y devuelva cuántas veces aparece d en n.

Ejemplo:
n = 707170, d = 7 → devuelve 3

⚠️ Prohibido usar bucles y strings.

Requisito extra: Complejidad temporal (Big-O).


## Ejercicio 10: Invertir un número (3 puntos)

Escribe una función recursiva estricta que reciba un entero positivo n y devuelva el número invertido.

Ejemplo:
n = 12030 → devuelve 3021 (sin ceros a la izquierda)

⚠️ Sin strings, sin bucles.

Requisito extra: Complejidad temporal y espacial (pila de recursión).
